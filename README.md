# OpenGL 3D Renderer

## Description 

This is an 3D ui library written in OpenGL


## Requirements

- Compiler that supports C++20
- CMake
- OpenGL Version 3.2 (minimum)

## Building

```shell
$ git clone https://github.com/abrka/opengl_ui_.git
$ cd opengl
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```
