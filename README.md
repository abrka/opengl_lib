# OpenGL Library

## Description 

This is an OpenGL abstraction library


## Requirements

- Compiler that supports C++20
- CMake
- OpenGL Version 3.2 (minimum)

## Building

```shell
$ git clone https://github.com/abrka/opengl_lib.git
$ cd opengl_lib
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```
